Philippine Red Cross Chapters
=============================

[Interactive map](http://philippineredcross.github.io/chapters/) showing Philippine Red Cross Chapters.

**Technology Used:**
- [Bootstrap v3.1.1](http://getbootstrap.com/)
- [Leaflet v0.7.2](http://leafletjs.com/)
- [Leaflet.markercluster](http://github.com/Leaflet/Leaflet.markercluster)
- [jQuery v1.11.0](https://ajax.googleapis.com/ajax/libs/jquery/1.11.0/jquery.min.js)
